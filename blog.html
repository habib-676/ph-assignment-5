Question-1: What are the different ways to select an element in the DOM?

We can select an element in the DOM in these ways:
1. document.getElementById(" ") --- this method will select the element by its id name.
2. document.getElementsByClassName(" ") --- this method will select elements by their class name.
3. document.getElementsByTagName(" ") --- this method will select elements by their tag name.
4. document.querySelector(" ")--- this method selects the first element that matches the selector name.
5. document.querySelectorAll(" ")--- this method will select all elements that match the selector name.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML :
- Syntax : varName.innerHTML = ` `;
- It allows to get or set HTML tags and contents inside an element. While innerText and textContent can not set any html tag.

innerText :
- Syntax : varName.innerText = " ";
- It gives all texts in the selected element. It shows only visible text in the selected element, not the html tags. It also allows to set the visible text in an element.

textContent :
- Syntax : varName.textContent = " ";
- It gives all text inside child elements. It shows only plaintext not the html tags. It is faster than innerText.

Question-3: What is event delegation in the DOM?

Event delegation allows to attach a single event listener, to a parent element, that will fire for all descendants matching a selector, whether those descendants exist now or are added in the future. Instead of writing multiple event listeners for every element, we can handle all events in one place.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a concept in the DOM (Document Object Model). It happens when an element receives an event, and that event bubbles up (or you can say is transmitted or propagated) to its parent and ancestor elements in the DOM tree until it gets to the root element.
The "Event Bubbling" behavior makes it possible for you to handle an event in a parent element instead of the actual element that received the event.
